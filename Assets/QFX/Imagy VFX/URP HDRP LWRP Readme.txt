--------------------------------------------------------------------------------------------------------------------------------
How to use in HDRP URP LWRP
--------------------------------------------------------------------------------------------------------------------------------

In order use the asset in the URP/HDRP/LWRP, you need to upgrade the materials: "Edit->Renderer Pipeline->Upgrade Materials"

Or you can import the package URP_Upgrade (since Unity 2019.3.15), HDRP_Upgrade (since Unity 2019.3.15)

--------------------------------------------------------------------------------------------------------------------------------

- URP/HDRP/LWRP are supported, except of the some distortion effects
- The Hole in the ground from the ShadowHole effect will not work in HDRP.

Name of the material: "P_Distortion"
Name of the effects: Rune of Power, Sakura Shield, Void Grenade, Lightning Splash, 
Dark Paralysis, Gunshots, Lightning Blast

Therefore, please just delete/disable particle system with the distortion material. (in the urp/hdrp path those particles are disabled).
