Please read the following instructions on how to add bloom and other effects:

For the correct work as in demo scene, you need to enable "HDR" on main camera and add correct bloom.

(Legacy Post Processing) !!!

The link on free unity physically is below;
https://www.assetstore.unity3d.com/en/#!/content/83912 

Use the following settings below;
Threshold 2
Radius 7
Intensity 1
High quality true
Anti flicker true

In the forward mode, HDR does not work with antialiasing. Therefore, you need to disable antialiasing (edit->project settings->quality) or use deferred rendering mode.
Add post processing behaviour(script) into your camera and choose PostProcessingProfile.asset in the IFX/PostProcessing folder.

---------------------------------------------------------------------------------------------------------

(Post Processing v2) !!!

Use the following settings below;

Bloom:
Intensity 5
Threshold 1

Color Grading: ACES
